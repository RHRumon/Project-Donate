Important FAQ

What is the Document Object Model (DOM)?

DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.

How do you select an element from the DOM?

To select a DOM element, we can target it by its ID, class, tag name, attribute or CSS selector.

What is event delegation in the context of the DOM and why it is useful?

When an event occurs on a child element, it "bubbles up" through its parent elements. By placing an event listener on a common ancestor, you can catch and handle events from any of its child elements without needing individual listeners on each one.

How do you manipulate an elements attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, we can access the element's attributes directly and modify them through properties like setAttribute, getAttribute, and removeAttribute for adding, reading, or removing attributes, respectively. For styles, we use the style property on the element, allowing we to set individual CSS properties in JavaScript, such as element.style.color = "blue";. Alternatively, we can add or remove entire classes using classList methods like add, remove, and toggle to apply predefined CSS rules. These techniques let we dynamically change an element's behavior, appearance, or content.
